**[Datasette Agent](https://simonwillison.net/2026/May/21/datasette-agent/#atom-everything)** | 2026-05-21 | 5/5
Simon Willison 发布 Datasette Agent 首个版本——为 Datasette 打造的可扩展 AI 助手，将他开发三年的 LLM Python 库与 Datasette 正式整合。核心设计是对话式数据查询：用户用自然语言提问，Agent 自动生成 SQL、执行查询、返回结果。架构上采用插件系统（charts 图表可视化、sprites 沙盒代码执行、openai-imagegen 图像生成），每个能力独立为插件。演示实例跑在 Gemini 3.1 Flash-Lite 上，成本低且足以胜任 SQLite 查询。这是垂直 agent 的典型范例——不做通用 chatbot，专注数据探索场景。
